Getting Start

  1. 依存ファイルとsensu.jsを読み込む

    <script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
<script src="//code.createjs.com/createjs-2015.05.21.min.js"></script>
<script src="./sensu.jquery.min.js"></script>

  2. canvasタグを用意

    <canvas id="sensu" />

  3. sensuメソッドを実行

    $(window).load(function(){
  $('#sensu').sensu({
    list: [
      {src: './img/demo_1.png', url: 'http://sawa-zen.com'},
      {src: './img/demo_2.png'},
      {src: './img/demo_3.png'}
    ]
  });
});

Settings

Option Type Default Description
list array [] 画像データの入ったリスト
roop boolean true ループさせるかどうか
autoplay boolean false 自動ページングをさせるかどうか
autoplaySpeed int 5000 自動ページングさせる時間
